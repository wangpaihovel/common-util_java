分支 M # 版本 ` M.0.0 ` 简介：

<p align="right">——2022年3月29日</p>

&emsp;&emsp;本项目提供了一些 Java 开发中常用的一些私人通用工具类。目前已包含如下工具。

* ID 生成器：用于生成 ID。
* JSON 序列化工具：提供用于序列化、反序列化 JSON 的工具。
* 多线程工具：用于方便地使用多线程技术将某些同步代码转化为异步代码。
* 资源读取：用于读取资源目录下的某些文件。
* 时间工具：用于获取本地、联网的北京时间等。
* 类型转化工具：用于对 JDK 中某些类型的转化。
